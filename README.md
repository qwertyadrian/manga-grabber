A simple manga/ranobe downloader written in Python

## Features (in progress)
- Download manga/ranobe from various sources
- Supports multiple formats (CBZ, PDF)
- Command-line interface

## Supported Sources
- [x] MangaLib
- [x] HentaiLib (need token to work)
- [x] RanobeLib
